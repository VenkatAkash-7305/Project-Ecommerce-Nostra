# Nostra – Ecommerce Website

Nostra is a responsive ecommerce front-end website built using **HTML, CSS, and JavaScript**.
The project demonstrates modern UI layout, product filtering, responsive navigation, and interactive components.

---

## 🚀 Live Demo

(After GitHub Pages deployment)

https://venkatakash-7305.github.io/Project-Ecommerce-Nostra/

---

## 📌 Features

* Responsive Navigation Bar
* Mobile Side Navigation Menu
* Product Collections Page
* Product Filtering (Occasion, Color, Arrivals)
* Search Bar UI
* Product Slider
* Most Wanted Section
* Like Button Toggle
* Newsletter Subscription Section
* Contact Page
* Responsive Layout for Mobile & Desktop

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts
* Font Awesome Icons

1. Clone the repository

```

```

2. Open the project folder

```
cd Project-Ecommerce-Nostra
```

3. Open **index.html** in your browser.
